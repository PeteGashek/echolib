Welcome to ECHOLIB

The goal of ECHOLIB is to provide a configuration driven utility for extending the functionality of the amazon echo for non-developers. It is extensible via components so that new features can be added easily without worrying about the Amazon Echo api.

This project is maintained by dwdamon
